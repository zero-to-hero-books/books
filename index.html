Zero-to-Hero: SQL

The Structured Query Language (SQL) is fairly straight forward. At the same time, seemingly simple search queries can be difficult to formulate in terms of SQL. Most books and tutorials center around using the where clause to search for a specific set of records in a table, putting the cart before the horse. In real world applications single table queries are rare. This book teaches not just how to query, but also how to think in terms of SQL. Starting from absolute zero, complex queries are built up as the book progresses. Starting with combining tables to grouping, aggregating and analytics with window functions. This book is intended to be concise and to the point, and teaches SQL and its usage from the end user perspective. Aspects of SQL that facilitate database administration and security such as access controls, roles, views, stored procedures and triggers are not covered.
